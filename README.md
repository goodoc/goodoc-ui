## Install

```
yarn add goodoc-ui

npm install --save goodoc-ui
```

## How to use

```
import { TextField } from "goodoc-ui"

function Component() {
  ...
  return (
    <SomeComponent>
      <TextField />
    <SomeComponent>
  )
}
```
